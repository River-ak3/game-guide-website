# 游戏攻略流量网站

基于 Just the Docs Jekyll 主题构建的游戏攻略网站。

## 在线访问

[https://river-ak3.github.io/game-guide-website](https://river-ak3.github.io/game-guide-website)

## 攻略内容

- 原神攻略 - 角色培养、深渊阵容、每日任务
- 王者荣耀攻略 - 上分英雄、团战技巧
- 我的世界攻略 - 新手入门、红石电路
- 艾尔登法环攻略 - 开局路线、BOSS弱点
- 塞尔达传说攻略 - 初始台地、四神兽

## 技术栈

- Jekyll 4.x
- Just the Docs 主题
- GitHub Pages 自动部署
- GitHub Actions CI/CD

## 本地运行

```bash
bundle install
bundle exec jekyll serve
```
