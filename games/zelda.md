# 塞尔达传说攻略

## 初始台地必做事项

1. 获取滑翔伞（完成4个神庙）
2. 制作料理恢复体力
3. 收集武器和防具
4. 学习基础战斗技巧

## 四神兽攻略

### 风·瓦·鲁达尼亚（火之神兽）
- 推荐装备：火焰套/攀岩套
- 核心机制：时间停止+炸弹投掷

### 水·露塔（水之神兽）
- 推荐装备：潜行套/游泳套
- 核心机制：控制水流方向

### 雷·瓦尔纳基（雷之神兽）
- 推荐装备：绝缘套
- 核心机制：过载金属物体

### 林·法伊尔（风之神兽）
- 推荐装备：飞行套/攀爬套
- 核心机制：操作风扇改变风向

## 人马击败技巧

1. 使用 mounts 反制（林克时间）
2. 远程弓箭射击马头
3. 炸弹箭造成大量伤害
4. 近身时盾反攻击
